🧮 Simple Calculator (Python)
📌 Description

This project is a Simple Calculator developed using Python. It provides a user-friendly graphical interface that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The calculator is designed to be simple, efficient, and easy to use, making it an excellent project for beginners learning Python GUI development.

The application is built using the Tkinter library, Python's standard GUI toolkit, and demonstrates the use of event-driven programming, functions, and graphical user interface design.

🚀 Features
➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division
🔢 Decimal number support
🧹 Clear button to reset calculations
⚠️ Error handling for invalid expressions
🖥️ Clean and user-friendly graphical interface
